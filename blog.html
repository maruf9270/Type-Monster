Difference between local storage and seassion storage:
Local storage:
Local storage is a storage in browser which can save some data of a website for a long priod of time. When user closes the browser the data does not wipe out by the browser. Local storage has 4 methods. They are: localStorage.setItem(key, value), localStorage.getItem(key), localStorage.removeItem(key), localStorage.clear()
Session Storage:
On the other hand sessoin storage data is instantly wiped out by the browser as soon as the page session ends. Session storage has four methods. They are: sessionStorage.setItem(key, value), sessionStorage.getItem(key), sessionStorage.removeItem(key), sessionStorage.clear()

Difference between global scope and block scope:
Global scope: The variables defined outside of any function or curly brackets are known as global variables and have global scope. Global scope means that the variables can be accessed from any part of that program, any function or conditional state can access that variable.
Block scope: Block scope is also a sub-type of local scope. The block scope can be defined as the scope of the variables inside the curly brackets {}. Now, these curly brackets can be of loops, or conditional statements, or something else. You are only allowed to refer to these variables from within the curly brackets {}.
